# 🧮 Mini Calculator

A simple calculator web app built using **HTML, CSS, and JavaScript**.  
Supports basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Features
- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🔢 Decimal support  
- 🧹 Clear button (C)  
- ⌫ Delete last input  
- ⚡ Real-time calculation with error handling
